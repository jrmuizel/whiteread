pub fn open<P: AsRef<Path>>(path: P) -> Result<WhiteReader<BufReader<File>>>

Opens a file, and wraps in WhiteReader

Shortcut for creating a file, wrapping it in a BufReader and then in a WhiteReader. This should be a static method on WhiteReader, but currently it's impossible to implement it that way (it requires equality constraints in where clauses).

Examples

Read an integer from a file.

let x: u32 = open("number.txt").unwrap().parse().unwrap();
